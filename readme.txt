Lazyest Stylesheet is a CSS editor that lets you customize your site design without modifying your theme or plugins.

== Description ==

Sometimes you need to tweak some style definitions in your themes or plugins. 
Lazyest Stylesheet stores your additional styles away from your themes and plugin directories.
You can safely update or change your themes and plugins without losing your hard work.

Lazyest Stylesheet features an editor with syntax highlighting. 
This plugin offers the Edit CSS functionality as in Jetpack, without installing the whole Jetpack package.

== Installation ==

1. Install using the WordPress plugin installer
2. Activate
3. Edit your permanent stylesheet in the menu: Appearance - Lazyest Stylesheet
